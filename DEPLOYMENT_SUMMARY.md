## 🚀 How to Deploy (Quick Start)

### 1️⃣ Prepare Files (5 minutes)
- Add your images to `images/` folder (compress to < 200KB each)
- Update contact information in `index.html`
- Customize colors in `css/style.css`
- Update foundation name and mission/vision

### 2️⃣ Deploy to Hostinger (15 minutes)
- Log in to Hostinger control panel
- Open File Manager
- Upload all files to `public_html` folder
- Maintain folder structure

### 3️⃣ Configure Domain (5 minutes)
- Go to Domains section
- Add/configure your domain
- Enable SSL/HTTPS certificate
- Wait for DNS propagation (24-48 hours)

### 4️⃣ Enable Performance (5 minutes)
- Enable GZIP compression
- Enable browser caching
- Test with Google PageSpeed Insights

### 5️⃣ Test & Launch (10 minutes)
- Visit your domain
- Verify all sections load
- Test mobile responsiveness
- Test on different devices
- Announce launch!

**⏱️ Total Time:** ~30 minutes for complete deployment
