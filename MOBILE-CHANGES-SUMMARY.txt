  HOW TO VIEW YOUR MOBILE WEBSITE
═══════════════════════════════════════════════════════════════════════════════

METHOD 1: BROWSER DEVTOOLS (Recommended - Instant!)
───────────────────────────────────────────────────────────────────────────────
1. Right-click index.html → Open with Chrome/Edge
2. Press F12 (Developer Tools)
3. Press Ctrl+Shift+M (Toggle device toolbar)
4. Select device: iPhone 12 Pro, iPhone SE, etc.
5. Test your mobile website!

METHOD 2: LOCAL SERVER (Test on Real Phone)
───────────────────────────────────────────────────────────────────────────────
1. Open PowerShell in project folder
2. Run: python -m http.server 8000
3. Get IP: ipconfig
4. On phone: http://YOUR_IP:8000

METHOD 3: DEPLOY ONLINE (Share with Others)
───────────────────────────────────────────────────────────────────────────────
• Netlify: Drag & drop folder → Get instant URL
• GitHub Pages: Upload to GitHub → Enable Pages
• Hostinger: Upload via FTP (already configured)
