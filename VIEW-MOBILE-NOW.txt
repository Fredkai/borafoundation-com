┌──────────────────────────────────────────────────────────────────────────────┐
│  METHOD 2: VIEW ON YOUR REAL PHONE                                          │
└──────────────────────────────────────────────────────────────────────────────┘

STEP 1: Start a local server
   → Open PowerShell in this folder
   → Type: python -m http.server 8000
   → Press Enter

STEP 2: Get your computer's IP address
   → In PowerShell, type: ipconfig
   → Look for "IPv4 Address" (e.g., 192.168.1.5)

STEP 3: Open on your phone
   → Make sure phone is on same WiFi
   → Open browser on phone
   → Type: http://YOUR_IP_ADDRESS:8000
   → Example: http://192.168.1.5:8000

🎉 Your website loads on your phone!

Q: Menu doesn't appear?
A: Make sure DevTools is in mobile view (Ctrl+Shift+M)
   The hamburger menu only shows on screens < 768px

Q: Can't see changes?
A: Press Ctrl+F5 to hard refresh the page

Q: Images don't load?
A: Check that all images are in the /images/ folder

Q: Layout looks broken?
A: Make sure you're using a modern browser (Chrome, Edge, Firefox)
